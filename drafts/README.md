# External Post Drafts

This directory holds public-safe drafts for posts that may be adapted to
external communities such as Planet Minecraft. These files are not linked from
the website navigation by default, but the repository itself is public, so every
draft must already satisfy the same scrub rules as the published site.

Before posting externally:

- Do not include server connection details, startup URLs, ports, internal
  infrastructure, exact sensitive coordinates, bot account details, or personal
  information.
- Treat external posting as a public release. Get explicit GO before posting or
  pushing anything that would make new pages visible.
- Prefer links to already-published public pages. If a linked page is only local
  or ahead of origin, publish the website first after explicit GO.
- Keep the post focused on ideas, screenshots, and public-readable stories, not
  operations.
